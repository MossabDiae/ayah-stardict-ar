# Ayah Index [Stardict dictionary]
### compatible with ***Goldendict***, StarDict, ..
  README available in [Arabic](README.ar.md)
  
A dictionary to search Quran for the corresponding ayah to a notation `(e.g 16:125)` and vice versa
